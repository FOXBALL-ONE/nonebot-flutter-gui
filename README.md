## ⚙️安装
我们推荐您前往``Release``页面下载对应的包

## 🔧手动构建
如果您想要手动构建，请确保你已经安装了``Flutter SDK``<br>

### 为Windows构建
```
flutter build windows
```
### 为Linux构建
```
flutter build linux
```
### 为MacOS构建
```
flutter build macos
```
